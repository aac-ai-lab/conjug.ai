Léxico verbos.json

conjugai-core · origem e atualização

O que é

Dados

O ficheiro vendors/conjugai-core/data/verbos.json guarda, por infinitivo (lema), paradigmas de cinco pessoas (0–4) para vários tempos e modos quando o MorphoBr tem paradigma completo — por exemplo presente, futuro, passado (pretérito perfeito), pretérito imperfeito, mais-que-perfeito, condicional, subjuntivos, imperativo, e ainda gerúndio, particípio (m/f × sg/pl) e infinitivo pessoal quando existem. O pipeline da demo CAA (analisarFrase) continua a usar só três tempos lógicos na deteção; conjugar na API aceita o conjunto alargado.

O JSON gerado pelo script oficial é minificado (sem indentação) para reduzir tamanho; o ficheiro pode ter dezenas de MB. O TypeScript importa o JSON em verbos-data.ts e o bundle assets/js/conjugai-core.js embute o léxico para uso offline no navegador.

Origem típica: MorphoBr

Apache-2.0

Uma forma de preencher ou alargar o léxico é usar o recurso público LR-POR/MorphoBr, que publica flexões em ficheiros verbs-*.dict (por letra do alfabeto).

No repositório ConjugAI, na raiz: npm run build:lexicon executa scripts/build_verbos_from_morphobr.sh, que descarrega (se necessário) os ficheiros verbs-a.dict … verbs-z.dict para scripts/cache/ e corre morphobr_dict_to_verbos.py para gerar data/verbos.json minificado. Opções como --whitelist, --require-core e merge com -m permitem léxico menor ou mais restrito.

Para descarregar só algumas letras, usa-se fetch_morphobr_verb_dicts.sh (variável LETRAS). O passo a passo e as etiquetas MorphoBr → JSON estão em vendors/conjugai-core/README.md (secção «Uso de dicionários linguísticos»).

Alternativa: CSV

Validação humana

Para dados revistos manualmente ou vindos de exportação (por exemplo fluxos DELAF/Unitex/Linguateca convertidos para tabela), usa-se vendors/conjugai-core/scripts/csv_to_verbos.py, que espera colunas lemma, form, tense, person (UTF-8).

Depois de alterar o JSON

Build

Na raiz do projeto: npm run build:lexicon quando atualizares os .dict ou o conversor; depois npm run build:core (ou npm run build) para regenerar assets/js/conjugai-core.js a partir do TypeScript e do verbos.json atual.

Documentação detalhada (comandos exactos, formato CSV, etiquetas MorphoBr): vendors/conjugai-core/README.md — secção «Uso de dicionários linguísticos».
